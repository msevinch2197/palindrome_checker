📘 README 1 — Palindrome Checker
🔁 Palindrome Checker (Matnni tekshirish)
📌 Loyiha haqida

Bu dastur foydalanuvchi kiritgan matn palindrome (oldidan va orqasidan bir xil o‘qiladigan) yoki yo‘qligini aniqlaydi.

⚙️ Qanday ishlaydi?
Matndan faqat harf va raqamlar olinadi
Katta-kichik harflar farqi yo‘q qilinadi
Matn teskari qilib solishtiriladi
▶️ Ishga tushirish
```
python palindrome.py
```

📌 Misollar
```
Matn: Madam
Natija: Bu palindrome ✅
```
```
Matn: Hello
Natija: Bu palindrome emas ❌
```
🧠 Xususiyatlari
Bo‘sh joy va belgilar e’tiborga olinmaydi
Katta/kichik harf farqi yo‘q
Har qanday matn bilan ishlaydi
🛠 Texnologiya
Python 3
